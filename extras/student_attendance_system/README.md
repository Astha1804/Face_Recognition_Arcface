# Student Attendance System using Face Recognition

This module demonstrates the application of the core face recognition engine for automated student attendance.
Components:

-Backend: Python + ArcFace recognition
-Frontend: Node.js-based interface
-Attendance marking via face recognition

# This is an extension built on top of the core face recognition system.
